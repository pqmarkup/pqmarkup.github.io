pqmarkup is a lightweight markup language (like Markdown, BBCode, Wiki markup, etc.), which is based on single paired quotation marks: characters 
‘
and 
’
.


Syntax of pqmarkup.


How to type paired single quotation marks (characters ‘ and ’)...
  • In Windows you can use without additional settings hotkey Alt+0145 (on the numeric keypad) for the opening quotation mark 
    ‘
    and hotkey Alt+0146 for the closing quotation mark 
    ’
    .
    By the way, this numbers on the numeric keypad are situated very felicitously:
    ┌───┬───┐
│ 4 │ 5 │
├───┼───┘
│ 1 │
├───┴───┐
│ 0     │
└───────┘
    It is similar to the left square bracket 
    [
    .
    So it's very easy to remember.
  • Using program AutoHotkey you can assign for example hotkeys Alt+9 and Alt+0 to insert single quotation marks by adding this two lines in your ahk-file:
    Alt & 9:: SendInput {‘}
Alt & 0:: SendInput {’}
  • In Sublime Text 3 (for Windows, Linux, or macOS): just install pqmarkup package.
  • In Android you can use Jbak Keyboard, which allows you to customize your keyboard layout.
    Here is a zip file with keyboard layouts with added characters ‘ and ’.
