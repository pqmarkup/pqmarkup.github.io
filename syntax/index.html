Is space necessary here? Why?
Yes, it is necessary.
Why — to avoid problems in following cases:
>‘Link text’[http://...] following text.

can be treated as:
>‘Link text’
[http://...] following text.

So, you should write:
> ‘Link text’[http://...] following text.

or:
>‘‘Link text’[http://...] following text.’
